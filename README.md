## Installation

```bash
$ npm install
```

## Criar banco com Compose

```bash
$ docker-compose up -d
```

## Derrubar o banco com Compose

```bash
docker-compose down -v
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
